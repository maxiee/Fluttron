# markdown_editor ui

Flutter Web UI for `examples/markdown_editor`.

Current scope:

- `MilkdownEditor` integration from `fluttron_milkdown`
- Runtime theme switching
- Save-state shortcut (`Cmd/Ctrl + S`)
- Status bar metrics

Build via project root:

```bash
dart run packages/fluttron_cli/bin/fluttron.dart build -p examples/markdown_editor
```
