# markdown_editor screenshot guide

Use this checklist when capturing screenshots for `examples/markdown_editor`.

## Capture scope

- Window size: `1280x800`
- Platform: macOS desktop
- Theme: `Nord` (default) unless demonstrating theme switch
- Data set: a folder with nested `.md` files and one unsaved file

## Required shots

1. Default launch state (welcome content, no folder opened)
2. Folder opened with sidebar populated
3. Unsaved indicator visible
4. Save As dialog for a new unsaved document
5. Theme switched (one dark theme screenshot)

## File naming

- `shot-01-launch.png`
- `shot-02-folder-opened.png`
- `shot-03-unsaved.png`
- `shot-04-save-as.png`
- `shot-05-dark-theme.png`

Store screenshots in this directory and reference them from
`examples/markdown_editor/README.md`.
