# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-14

### Added
- Initial skeleton with MilkdownEditor widget
- Crepe-based markdown editor with CommonMark support
- Event system: `change`, `ready`, `focus`, `blur`
- Config normalization for `initialMarkdown`, `theme`, `readonly`
- CSS isolation with `fluttron-milkdown` BEM prefix
